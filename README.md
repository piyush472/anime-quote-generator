# Random Anime Quote Generator

A simple web application built with **Flask** that displays a random anime quote each time the page is refreshed or the user clicks the generate button.

The application fetches quotes from a public anime quote API and displays the quote along with the character name and anime title.

## Features

* Generate random anime quotes
* Displays character and anime name
* Simple and clean interface
* Fast API-based quote retrieval

## Tech Stack

* Python
* Flask
* HTML
* CSS
* REST API

## Installation

1. Clone the repository

```
git clone https://github.com/yourusername/anime-quote-generator.git
```

2. Navigate to the project directory

```
cd anime-quote-generator
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the application

```
python app.py
```

5. Open in your browser

```
http://127.0.0.1:5000
```
